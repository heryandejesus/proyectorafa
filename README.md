# Rafael Scalp Artist — Scalp Micropigmentation Website

Professional business website built for a scalp micropigmentation specialist
based in Buenos Aires, Argentina.

## 🔗 Live Site

[heryandejesus.github.io/proyectorafa](https://heryandejesus.github.io/proyectorafa/)

## 📋 About

A multi-section website for a scalp micropigmentation artist, designed to showcase
services, training programs, client testimonials, and before/after results.
Includes WhatsApp contact integration and a contact form.

## ✨ Features

- Hero section with background video
- Services showcase (Efecto Afeitado, Rapado, Línea Frontal, Densidad, Cicatrices)
- Before & after results gallery
- Client testimonials section
- Professional training / courses section
- WhatsApp floating button for direct contact
- Contact form
- Fully responsive design

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
